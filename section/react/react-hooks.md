# React Hooks

#### Hooks

* 갈고리 처럼 연결하다, 낚아채다
* 재사용 가능한 로직들을 연결한다. (리액트에서 제공한 기능, 혹은 직접 만들 커스텀 훅)
* 함수형 컴포넌트를 사용할 때 사용하며 상태(useState) 및 라이프 사이클(useEffect) 관리할 수 있도록 도운다.
* `useState, useEffect, useMemo...` 등등이 있고 커스텀 할 때도`use..`로 시작

&#x20;
