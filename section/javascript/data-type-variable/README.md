# Data Type Variable

#### 변수 할당

* `Number, String, Boolean, Undefined, Null, Symbol` 과 같은 원시 타입 데이터를 할당하게 되면 데이터 자체 할당.
* `object/reference type`과 같은 참조 타입이 할당되면 object 메모리가 들어있는 주소값(참조값)이 할당.
